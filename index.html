TumblrBuilder

Javascript library to build custom Tumblr from API

Get Started →

Default and custom theme

All pages and article types have a minimalist and functionnal default theme with popular fields and can individually be overrides

Extra features

Home and tagged page have a native infinite scroll, get the list of all hashtags used by your articles, get related posts and use browser cache to increase load performance

Inside or outside of Tumblr

TumblrBuilder can works inside Tumblr blog and outside in your personal website, thanks to the API

# Easy install

npm install tumblr-builder
